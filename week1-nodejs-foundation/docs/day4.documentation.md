# Day 4: Async JavaScript

## What I Did
- Learned sync vs async
-  Used callbacks to read files
- Used promises
-  Used async/await
-  Read multiple files with Promise.all
-  Made API call with fetch

## Key Learnings

### Async/Await is Best
- Easy to read
- Looks like normal code
- Must use try/catch for errors

### Always Use await
- Without await = get Promise, not data
- With await = get actual value


## Common Mistakes
- Forgetting await
- Not handling errors
