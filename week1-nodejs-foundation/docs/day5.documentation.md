# Day 5: CLI Note-Saver Project

## What I Built
A command-line tool to save, view, and delete notes

## Features
-  Add notes with timestamp
-  List all notes
-  Delete notes by ID
-  Delete all notes
-  Auto-creates file
-  Error handling

## How to Use
node notes.js add "My note"
node notes.js list
node notes.js delete 1
