DAY 5: Role-Based Access Control

WHAT I DID:
- Added role field to User model
- Created authorise middleware
- Admin can see all users
- Users can only update own profile

ROLES:
- user: Regular user
- admin: Full access

MIDDLEWARE CHAIN:
router.get('/', protect, authorise('admin'), handler)

TESTED:
✅ Admin can see all users
✅ Regular user cannot
✅ Users can update own profile only
✅ Admin can delete any user
